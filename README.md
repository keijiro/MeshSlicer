# MeshSlicer

**MeshSlicer** is a Unity sample project that explores implementing a mesh slicing
feature using Unity MCP Server and external AI coding agents.

- Demo: Contains an interactive demo project.
- Work: Contains a design prompt, unit tests, and benchmark results.
